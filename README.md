# Репозиторий проекта "Мини-Склад"
Создана конфигурация с минимальным контуром учёта:

Справочники: Товары, Контрагенты, Сотрудники, Склады

Документы: Поступление, Реализация, Ввод начальных остатков, Начисления сотрудникам 

Интерфейс: добавлены объекты в подсистемы «Мини-Склад», "Бухгалтерия", выведены команды на панель

Регистры накопления: Остатки товаров, Продажи, Остатки товаров по партиям, Стоимость товаров

Регистры сведений: Цены, Графики работы, Склады пользователей

СКД-отчет Обороты по товарам, Остатки товаров на дату, Оборотно-сальдовая ведомость, Перерасчет, Начисления сотрудникам

Печатная форма документа Реализация, форма с подбором товаров

План счетов, регистр бухгалтерии, регистр расчета

Регламентные задания по автоматическому обновлению полнотекстового индекса

Созданы роли: Администратор, Бухгалтер, Менеджер, Кладовщик, Гость и назначены права для каждой

Созданы правила обмена между центральной базой и филиалом, между центральной базой и отделением.
