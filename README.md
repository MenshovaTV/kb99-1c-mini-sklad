# Репозиторий проекта "Мини-Склад"
Создана конфигурация с минимальным контуром учёта:

Справочники: Товары, Контрагенты.

Документы: Поступление, Реализация.

Интерфейс: добавлены объекты в подсистему «Мини-Склад», выведены команды на панель

Регистры: Остатки товаров, Продажи, Остатки товаров по партиям, Стоимость товаров

СКД-отчет Обороты по товарам, Остатки товаров на дату

Печатная форма документа Реализация
